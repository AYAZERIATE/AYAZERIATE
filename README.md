## &nbsp;About

I'm a full-stack developer and final-year **Digital Development (Web Full Stack)** student at **ISTA Hay Riad (OFPPT)**, Rabat — I design and ship complete web products, from database schema to responsive UI.

I'm currently completing an internship at Morocco's **Ministère de l'Équipement, du Transport et de l'Eau**, building the frontend of **TrackRoute**, a logistics/route-tracking platform, with a Laravel 11 + React/Vite stack and JWT/Sanctum authentication.

- 🛠️ Building **TrackRoute** — a full-stack tracking platform (Laravel 11 API + React/Vite frontend, dark cyber/HUD design system)
- 🤖 Building **AI Success Planner** — a Python CLI career assistant powered by a local Ollama (llama3.2) model
- 🎯 Refactored **MADLIBS_GAME**, a 10-level goal-tracking & motivation app, into a clean layered architecture (`services/`, `utils/`, `models/`)
- 📚 Studied REST APIs, MVC, Agile/Scrum, UML/Merise, and MySQL/MongoDB as part of my formation
- 🌱 Actively applying for Full-Stack Developer internships in Rabat

**Open to:** Full-Stack Developer Internships · Frontend Developer Roles
